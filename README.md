# Pinterest Bulk Publisher

A professional web application for scheduling and publishing multiple pins to Pinterest efficiently.

## 🎯 Features

- **Bulk Image Upload**: Upload up to 300 images at once
- **Smart Scheduling**: Schedule pins with custom time intervals
- **CSV Import**: Import pin data from CSV files
- **Sequential Matching**: Auto-match images (PIN1000 → Pin 1, PIN1001 → Pin 2, etc.)
- **Safe Mode**: Random delays to prevent account restrictions
- **Real-time Progress**: Monitor publishing status with detailed logs

## 🚀 How to Use

1. Upload your CSV file containing pin titles and descriptions
2. Select the folder containing your images (PIN1000 to PIN1299)
3. Choose your target Pinterest board
4. Set scheduling interval (recommended: 10-15 minutes)
5. Start publishing!

## 🛡️ Safety Features

- Compliant with Pinterest Terms of Service
- Rate limiting to avoid API restrictions
- Random delays between pins (30-90 seconds)
- Error handling and retry mechanisms

## 📋 Requirements

- Pinterest Business Account
- Modern web browser (Chrome, Firefox, Safari)
- Images named sequentially (e.g., pin_image_PIN1000.jpg)
